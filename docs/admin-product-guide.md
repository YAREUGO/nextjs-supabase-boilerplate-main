# 어드민 상품 등록 가이드

이 문서는 Supabase 대시보드에서 직접 상품을 등록하는 방법을 안내합니다.

## 접근 방법

1. [Supabase 대시보드](https://app.supabase.com)에 로그인합니다.
2. 프로젝트를 선택합니다.
3. 왼쪽 메뉴에서 **Table Editor**를 클릭합니다.
4. `products` 테이블을 선택합니다.

## 상품 등록 방법

### 1. 새 상품 추가

1. `products` 테이블에서 **Insert** 버튼을 클릭합니다.
2. **Insert row** 옵션을 선택합니다.
3. 다음 필드를 입력합니다:

| 필드명 | 타입 | 필수 | 설명 | 예시 |
|--------|------|------|------|------|
| `name` | TEXT | ✅ | 상품명 | "면 100% 기본 티셔츠" |
| `description` | TEXT | ❌ | 상품 설명 | "심플한 디자인, 5가지 컬러" |
| `price` | DECIMAL(10,2) | ✅ | 가격 (0 이상) | 25000 |
| `category` | TEXT | ❌ | 카테고리 | "clothing" |
| `stock_quantity` | INTEGER | ❌ | 재고 수량 (기본값: 0) | 300 |
| `is_active` | BOOLEAN | ❌ | 활성화 여부 (기본값: true) | true |

### 2. 카테고리 값

사용 가능한 카테고리 값:
- `electronics` - 전자제품
- `clothing` - 의류
- `books` - 도서
- `food` - 식품
- `sports` - 스포츠
- `beauty` - 뷰티
- `home` - 생활/가정

### 3. 자동 생성 필드

다음 필드는 자동으로 생성되므로 입력할 필요가 없습니다:
- `id` - UUID (자동 생성)
- `created_at` - 생성일시 (자동 생성)
- `updated_at` - 수정일시 (자동 갱신)

## 상품 수정 방법

1. `products` 테이블에서 수정할 상품의 행을 클릭합니다.
2. 수정할 필드를 변경합니다.
3. **Save** 버튼을 클릭합니다.

**참고**: `updated_at` 필드는 자동으로 현재 시간으로 갱신됩니다.

## 상품 삭제 방법

1. `products` 테이블에서 삭제할 상품의 행을 선택합니다.
2. **Delete** 버튼을 클릭합니다.
3. 확인 메시지에서 **Confirm**을 클릭합니다.

**주의**: 상품을 삭제하면 관련된 장바구니 아이템도 함께 삭제됩니다 (CASCADE 설정).

## 상품 비활성화

상품을 삭제하지 않고 일시적으로 숨기려면:
1. 해당 상품의 `is_active` 필드를 `false`로 변경합니다.
2. 이렇게 하면 상품 목록에 표시되지 않지만 데이터는 유지됩니다.

## 샘플 데이터

개발용 샘플 데이터는 `supabase/migrations/update_shopping_mall_schema.sql` 파일의 마지막 부분에 포함되어 있습니다. 마이그레이션을 실행하면 자동으로 20개의 샘플 상품이 추가됩니다.

## 주의사항

1. **가격**: 소수점 2자리까지 입력 가능합니다 (예: 25000.50)
2. **재고**: 음수는 입력할 수 없습니다 (CHECK 제약조건)
3. **카테고리**: 대소문자를 구분합니다. 정확한 값을 입력하세요.
4. **이미지**: 현재 MVP에서는 이미지 필드가 없습니다. Phase 2 이후에 추가될 예정입니다.

## 문제 해결

### 상품이 목록에 표시되지 않는 경우

1. `is_active`가 `true`인지 확인하세요.
2. `stock_quantity`가 0보다 큰지 확인하세요 (인기 상품 섹션의 경우).

### 가격이 올바르게 표시되지 않는 경우

1. `price` 필드가 숫자 형식인지 확인하세요.
2. 소수점은 점(.)으로 구분하세요 (예: 25000.50).

### 카테고리가 표시되지 않는 경우

1. 카테고리 값이 정확한지 확인하세요 (대소문자 구분).
2. 카테고리 라벨이 `types/product.ts`의 `CATEGORY_LABELS`에 정의되어 있는지 확인하세요.
