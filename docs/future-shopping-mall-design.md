# 10년 뒤 쇼핑몰 - 라이프맵 기반 인터랙티브 쇼핑몰 설계안

## 전체 컨셉

**"사용자의 하루 타임라인 위에 상품을 배치하는 인터랙티브 라이프맵 쇼핑몰"**

- 카테고리(도서/식품/스포츠/의류) 중심 ❌
- 시간·장면 기준 상품 묶음 ✅
- 개별 상품 ❌ → 시나리오 카드(루틴 세트) ✅

---

## 섹션 1: "오늘의 라이프맵" (히어로 섹션)

### 레이아웃 구조

**전체 구조: 좌우 분할 레이아웃 (Desktop) / 세로 스택 (Mobile)**

#### 좌측 영역 (30% 너비, 고정)
- **세로 타임라인 바**
  - 높이: 100vh (뷰포트 전체 높이)
  - 배경: 그라데이션 (새벽 → 아침 → 낮 → 저녁 → 밤)
  - 타임라인 마커: 현재 시간대 하이라이트 (애니메이션 펄스 효과)
  - 각 시간대 라벨: "새벽 5시", "아침 7시", "오전 9시", "점심 12시", "오후 3시", "저녁 6시", "밤 9시"
  - 스크롤 시 타임라인은 고정, 우측 콘텐츠만 스크롤

#### 우측 상단 영역 (70% 너비, 스크롤)
- **히어로 카드 영역**
  - 높이: 60vh
  - 배경: 현재 시간대에 맞는 그라데이션
    - 새벽: `from-indigo-900 via-purple-900 to-pink-900`
    - 아침: `from-orange-200 via-yellow-200 to-amber-200`
    - 낮: `from-blue-400 via-cyan-400 to-teal-400`
    - 저녁: `from-purple-500 via-pink-500 to-orange-500`
    - 밤: `from-slate-800 via-slate-900 to-indigo-900`
  - 카드 배치: 3D tilt 효과 (CSS transform: perspective)
  - 상품 이미지 플로팅: `position: absolute`로 배경에 배치, `opacity: 0.3`, `blur: 20px`

#### 우측 하단 영역 (70% 너비, 스크롤)
- **시나리오 선택 영역**
  - 높이: 40vh
  - 가로 스크롤 카드 그리드
  - 각 카드: 300px 너비, 400px 높이
  - 카드 간격: 24px

### 핵심 카피 예시

**메인 헤드라인:**
```
오늘, 이렇게 살아볼까요?
```

**서브 헤드라인:**
```
지금은 [현재 시간]시, [시간대명] 시간이에요
```

**시나리오 카드 예시:**
- "출근 전 30분 루틴" - 커피 + 책 + 운동화
- "점심시간 1시간 리셋" - 샐러드 + 에어팟 + 책
- "퇴근 후 1시간 리셋" - 운동복 + 보조배터리 + 음료
- "주말 오전 집중 루틴" - 노트북 + 커피 + 의자

### 인터랙션 설명

1. **타임라인 클릭**: 해당 시간대로 이동, 우측 히어로 배경색 변경
2. **시나리오 카드 호버**: 카드 확대 (scale: 1.05), 그림자 강화
3. **시나리오 카드 클릭**: 해당 루틴 상세 페이지로 이동
4. **스크롤**: 타임라인 고정, 우측만 스크롤, 스크롤 위치에 따라 타임라인 마커 업데이트

## 섹션 2: "사람들이 자주 고르는 루틴"

### 레이아웃 구조

**전체 구조: 그리드 레이아웃 (3열 Desktop, 2열 Tablet, 1열 Mobile)**

- **그리드 컨테이너**
  - `grid-cols-1 md:grid-cols-2 lg:grid-cols-3`
  - Gap: 32px
  - Padding: 80px (상하), 40px (좌우)

- **루틴 세트 카드**
  - 너비: 100% (그리드에 맞춤)
  - 높이: 500px
  - 구조:
    - 상단 (60%): 히어로 이미지 영역 (그라데이션 배경 + 플로팅 상품 아이콘)
    - 하단 (40%): 텍스트 영역
      - 루틴 제목: 24px, bold
      - 루틴 설명: 16px, regular
      - 포함 상품 수: 14px, muted
      - CTA 버튼: "이 루틴 담기" / "상품만 골라 담기"

### 핵심 카피 예시

**섹션 제목:**
```
사람들이 자주 고르는 루틴
```

**서브 제목:**
```
실제로 매일 쓰는 조합만 모았어요
```

**루틴 카드 예시:**

1. **"아침 7시, 출근 준비 루틴"**
   - 설명: "커피 한 잔 + 오디오북 + 깔끔한 옷차림"
   - 포함 상품: 커피믹스, 무선 이어폰, 기본 티셔츠 (3개)

2. **"점심 12시, 데스크 루틴"**
   - 설명: "건강한 식사 + 집중용 음악 + 눈 피로 완화"
   - 포함 상품: 샐러드, 블루라이트 차단 안경, 스탠딩 데스크 (3개)

3. **"저녁 7시, 운동 루틴"**
   - 설명: "운동복 + 보조배터리 + 운동 후 보충제"
   - 포함 상품: 레깅스, 보조배터리, 프로틴 (3개)

### 인터랙션 설명

1. **카드 호버**: 상단 이미지 확대 (scale: 1.1), 하단 텍스트 영역에 그림자 추가
2. **"이 루틴 담기" 클릭**: 루틴 내 모든 상품이 장바구니에 추가, 토스트 알림 표시
3. **"상품만 골라 담기" 클릭**: 모달 열림, 루틴 내 상품 목록 표시, 체크박스로 선택 가능

## 섹션 3: "카테고리 대신 장면으로 보기"

### 레이아웃 구조

**전체 구조: 탭 네비게이션 + 그리드 레이아웃**

- **탭 네비게이션 (상단 고정)**
  - 가로 스크롤 가능한 탭 버튼들
  - 각 탭: "🏠 집", "🏢 회사", "💪 헬스장", "☕ 카페", "🚗 이동", "🌙 집 (저녁)"
  - 활성 탭: 하단 border 3px, 색상 강조
  - 탭 간격: 16px

- **씬 필터 영역 (탭 아래)**
  - 높이: 200px
  - 배경: 선택된 장면에 맞는 그라데이션
  - 중앙 텍스트: "이 장면에서 필요한 것들"
  - 플로팅 아이콘: 선택된 장면 관련 상품 아이콘들이 배경에 떠다님

- **상품 그리드 (하단)**
  - `grid-cols-2 md:grid-cols-4 lg:grid-cols-6`
  - Gap: 24px
  - 각 상품 카드: 정사각형 (1:1 비율)
  - 카드 내부:
    - 상단 80%: 상품 이미지/아이콘
    - 하단 20%: 상품명 (1줄, 말줄임)

### 핵심 카피 예시

**섹션 제목:**
```
카테고리 대신 장면으로 보기
```

**서브 제목:**
```
어디서, 언제 쓰는지가 더 중요해요
```

**씬별 카피:**

- **🏠 집 (아침)**: "아침을 시작하는 공간"
- **🏢 회사**: "업무에 집중하는 공간"
- **💪 헬스장**: "몸을 움직이는 공간"
- **☕ 카페**: "휴식과 영감의 공간"
- **🚗 이동**: "이동 중에도 편하게"
- **🌙 집 (저녁)**: "하루를 마무리하는 공간"

### 인터랙션 설명

1. **탭 클릭**: 해당 장면으로 필터링, 배경 그라데이션 변경, 상품 그리드 업데이트
2. **상품 카드 호버**: 카드 확대 (scale: 1.05), 상품명 표시
3. **상품 카드 클릭**: 상품 상세 페이지로 이동

## 섹션 4: "LAB 리포트 (실험 데이터)"

### 레이아웃 구조

**전체 구조: 대시보드 스타일 카드 레이아웃**

- **그리드 컨테이너**
  - `grid-cols-1 md:grid-cols-2 lg:grid-cols-3`
  - Gap: 24px

- **실험 결과 카드**
  - 너비: 100%
  - 높이: 300px
  - 배경: 반투명 유리 효과 (backdrop-blur)
  - 구조:
    - 상단: 아이콘 + 제목
    - 중앙: 숫자/차트 (대형)
    - 하단: 설명 텍스트

### 핵심 카피 예시

**섹션 제목:**
```
LAB 리포트
```

**서브 제목:**
```
실제 사용자 데이터로 검증한 결과예요
```

**실험 결과 카드 예시:**

1. **"재구매 실험"**
   - 아이콘: 🔄
   - 숫자: "85%"
   - 설명: "같은 루틴을 다시 사는 사람들"

2. **"만족도 실험"**
   - 아이콘: ⭐
   - 숫자: "4.8/5.0"
   - 설명: "실사용자 평균 평점"

3. **"리셋 실험"**
   - 아이콘: 🔄
   - 숫자: "30일"
   - 설명: "무료 교환 보장 기간"

4. **"조합 실험"**
   - 아이콘: 🧪
   - 숫자: "1,234개"
   - 설명: "검증된 루틴 조합 수"

5. **"시간 실험"**
   - 아이콘: ⏰
   - 숫자: "평균 2분"
   - 설명: "루틴 선택 소요 시간"

6. **"성공 실험"**
   - 아이콘: ✅
   - 숫자: "92%"
   - 설명: "루틴 완료율"

### 인터랙션 설명

1. **카드 호버**: 카드 상승 (translateY: -8px), 그림자 강화
2. **카드 클릭**: 상세 리포트 모달 열림 (선택사항)

## 섹션 5: "내 하루 시나리오 만들어보기" (CTA)

### 레이아웃 구조

**전체 구조: 중앙 정렬, 대형 CTA 버튼**

- **컨테이너**
  - 높이: 400px
  - 배경: 그라데이션 (from-blue-600 via-purple-600 to-pink-600)
  - 중앙 정렬: flex items-center justify-center

- **CTA 영역**
  - 중앙 배치
  - 구조:
    - 상단: 큰 제목 (48px)
    - 중앙: 설명 텍스트 (20px)
    - 하단: CTA 버튼 (대형, 60px 높이)

### 핵심 카피 예시

**제목:**
```
내 하루 시나리오 만들어보기
```

**설명:**
```
시간대별로 필요한 것들을 조합해서
나만의 루틴을 만들어보세요
```

**버튼:**
```
시나리오 만들기 시작하기
```

### 인터랙션 설명

1. **버튼 호버**: 버튼 확대 (scale: 1.05), 그림자 강화
2. **버튼 클릭**: 시나리오 빌더 페이지로 이동

## 기술 스택 제안

- **레이아웃**: Tailwind CSS Grid + Flexbox
- **애니메이션**: Framer Motion (React) 또는 CSS Transitions
- **인터랙션**: React Hooks (useState, useEffect)
- **타임라인**: Custom SVG 또는 Canvas
- **3D 효과**: CSS Transform (perspective, rotateX)

---

## 구현 우선순위

1. **Phase 1**: 섹션 1 (히어로) + 섹션 2 (루틴) - 핵심 기능
2. **Phase 2**: 섹션 3 (장면 필터) - 필터링 기능
3. **Phase 3**: 섹션 4 (LAB 리포트) + 섹션 5 (CTA) - 신뢰도 강화

---

## 디자인 톤 가이드

- **색상**: 그라데이션 중심 (시간대별 색상 변화)
- **타이포그래피**: Sans-serif (현대적, 깔끔)
- **간격**: 넉넉한 여백 (breathing room)
- **애니메이션**: 부드럽고 자연스러운 전환 (ease-in-out)
- **유리 효과**: backdrop-blur로 미래지향적 느낌
